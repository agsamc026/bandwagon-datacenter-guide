## 搬瓦工"机房"到底指什么

搬瓦工（BandwagonHost）是美国 IT7 旗下的 VPS 服务商，目前在美国、日本、香港、荷兰、加拿大、新加坡、澳大利亚、迪拜等地部署了超过 25 个数据中心。买 VPS 之后，服务器就运行在其中某个机房里。

所谓"换机房"，是指通过 KiwiVM 控制面板把你的 VPS 从一个数据中心迁移到另一个数据中心——IP 会变，数据会保留（一般情况下）。这个功能叫"Automatic migration between datacenters"，产品描述页上写了才支持。

## 不同套餐的可选机房数量差异

这是核心。搬瓦工目前主要有四个套餐系列，可选机房数量从 1 个到 15+ 个不等。

**CN2 GIA ECOMMERCE（CN2 GIA-E）套餐**：可选机房最多，12 个以上，包含 5 个 CN2 GIA 机房以及日本软银、荷兰联通等优质线路机房。这是搬瓦工性价比最高的系列，$169.99/年起（季付 $49.99）。

**香港 / 日本大阪 CN2 GIA 套餐（Ultra 系列）**：香港套餐可选 7 个机房，含香港、新加坡、日本东京、日本大阪这四个亚太高端机房，加上美国 CN2 GIA 机房。延迟低，价格贵，$899.99/年起。

**香港套餐的特殊限制**：香港机房只进不出——买了香港套餐可以迁到其他机房，但普通套餐的 VPS 没法迁入香港机房。

## 不同运营商应该选哪个机房

这个问题跟"可以有几个机房"是连着的——买了 CN2 GIA-E 套餐之后，该停在哪个机房才对？

**电信用户**：DC6 和 DC9 都很合适，回程走 CN2 GIA，晚高峰表现稳。DC6 的带宽冗余容量通常更足，DC9 的硬件已经升级到 AMD EPYC + NVMe，速度更快。两个都可以试试，哪个当下 ping 值低就停哪个。

**联通用户**：同样推荐 DC6 或 DC9，回程 CN2 GIA，表现接近电信。荷兰 EUNL\_9 对联通也不错，延迟高一些但带宽稳定。

**移动用户**：DC6/DC9 回程走 CMIN2，这是移动国际专线，比之前的软银或 CN2 绕行好很多。日本大阪软银机房对部分移动用户也有不错表现，但晚高峰偶尔会有波动，不如直接选有 CMIN2 的美国机房稳。

讲真，买了 CN2 GIA-E 套餐之后最省事的方式是：先在 DC6 或 DC9 跑几天，感觉不对就直接在 KiwiVM 后台一键切换，不需要重新买，也不扣流量。切换时 VPS 会有几分钟停机，换完 IP 换了就行，数据不丢。

## KVM 普通套餐和 CN2 GIA-E 的机房区别，要不要升级

顺便说一下这个。KVM 普通套餐不含 CN2 GIA 机房，走的是普通线路（DC8 ZNET、DC2 QNET 等），延迟相比 CN2 GIA 高一些，高峰期更明显。

如果你只是跑个脚本、学学 Linux 命令，KVM 套餐完全够用，一年 $49.99 省得多。但如果是建站、部署需要访问的服务，或者对延迟有感知，那 CN2 GIA-E 才是该选的。算下来 CN2 GIA-E 最低季付 $49.99，年付 $169.99，折合每天不到三毛钱，和 KVM 套餐的价差没那么夸张。

## 常见问题

**Q：买 CN2 GIA-E 套餐后，机房迁移有次数限制吗？**

没有明确次数限制，但每次迁移之间有冷却时间（一般约 72 小时），不能频繁切换。

**Q：切换机房之后 IP 会变吗？**

会变。迁移完成后分配一个新机房的 IP，原来的 IP 不保留。如果原来的 IP 被封，迁移机房也是顺便换 IP 的常用方法。

**Q：搬瓦工有多少个机房？**

目前总机房数超过 25 个，分布在美国、日本、香港、荷兰、加拿大、新加坡、澳大利亚、迪拜等地。但不是所有套餐都能用所有机房，能用的机房范围取决于套餐类型。

**Q：30 天内不满意可以退款吗？**

可以。搬瓦工对新用户提供 30 天内全额退款，不满意直接申请，没有合同约束。

**Q：KVM 套餐能换机房吗？**

能换，可以在几个美国和荷兰的普通线路机房之间切换，但不能迁入 CN2 GIA 机房。
